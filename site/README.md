# Spattel — сайт

Лендинг «ремонт под ключ без тревоги» + страница для партнёров/бригад.
Каждая страница — самодостаточный HTML-файл: CSS и JS вшиты в `<style>`/`<script>`
внутри самого файла, шрифты — с Google Fonts по CDN. Сборки и зависимостей нет.

## Структура

```
site/
├── index.html              # главная страница (лендинг, конструктор сметы, FAQ)
├── spattel_partners.html   # страница для партнёров/бригад
├── 404.html                 # страница «не найдено»
├── robots.txt
└── sitemap.xml
```

## Перед публикацией — обязательно заменить

- [ ] Реальный телефон и email (сейчас плейсхолдеры `+7 (000) 000-00-00` / `hello@spattel.ru`)
- [ ] Подключить реальную отправку формы/заявки
- [ ] Домен в `<link rel="canonical">`, Open Graph тегах, `robots.txt` и `sitemap.xml`

## Деплой

### Vercel
Проект подключён к GitHub-репозиторию `svtomilov/Spattel1`: пуш в ветку `main`
триггерит сборку и публикацию автоматически. Root Directory в настройках
Vercel-проекта — `site`, build command не нужен (статика).

Ручной деплой (если нужно):
1. `npm i -g vercel` (если не установлен)
2. В папке `site/` выполнить `vercel` и следовать подсказкам
3. Для продакшна: `vercel --prod`

### Любой статический хостинг
Просто скопировать содержимое `site/` на сервер — зависимостей и шага сборки нет.
