# Spattel — сайт

Одностраничный лендинг «Капитальный ремонт вторички под ключ с личным руководителем проекта».
Чистый HTML/CSS/JS, без фреймворков и сборки. Анимации на скролле — GSAP + ScrollTrigger (CDN).

## Структура

```
site/
├── index.html          # главная страница
├── 404.html             # страница «не найдено»
├── robots.txt
├── sitemap.xml
├── css/style.css
├── js/main.js           # скролл-скраб hero-видео, reveal-анимации, форма лида, мобильная sticky-CTA
└── assets/
    ├── favicon.svg
    ├── hero-transform.mp4      # видео «убитая вторичка → лофт», скраб по скроллу в #hero3d
    └── placeholder-photo.svg   # временная заглушка вместо реальных фото
```

## Перед публикацией — обязательно заменить

- [ ] Реальный телефон, email, мессенджеры (сейчас плейсхолдеры `+7 (900) 000-00-00` / `hello@spattel.ru`)
- [ ] Реквизиты ИП в футере (`ИП Фамилия Имя Отчество · ИНН 000000000000`)
- [ ] Домен в `<link rel="canonical">`, Open Graph тегах, `robots.txt` и `sitemap.xml` (сейчас `https://spattel.ru/` как заглушка)
- [ ] Фото объектов «до/после» вместо `assets/placeholder-photo.svg` в hero и в разделе «Наши работы»
- [ ] Подключить реальную отправку формы лида (сейчас `js/main.js` только валидирует телефон и показывает сообщение об успехе — TODO с местом для интеграции отмечен в коде)
- [ ] Проверить, что цифры социального доказательства (4 года, 120+ объектов) актуальны

## Деплой

### Vercel
1. `npm i -g vercel` (если не установлен)
2. В папке `site/` выполнить `vercel` и следовать подсказкам — root directory уже статический, build command не нужен
3. Для продакшна: `vercel --prod`

### Netlify
1. Перетащить папку `site/` в Netlify Drop (app.netlify.com/drop), либо
2. Подключить репозиторий, указать **Base directory:** `site`, **Publish directory:** `site` (build command — пусто, это статика)

### Любой статический хостинг
Просто скопировать содержимое `site/` на сервер — зависимостей и шага сборки нет.
