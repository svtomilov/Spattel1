# Spattel — аудит качества перед сдачей (Фаза 6)

Проверка сайта в `site/` перед публикацией. Проверено визуально через headless Chrome
(скриншоты при ширине 600–1440px — стабильно и без искажений) и разбором исходников.
Отдельно отмечено ограничение методики ниже.

**Ограничение метода:** в этом окружении нет Lighthouse/PageSpeed и нет мобильного браузера —
проверка на реальном телефоне не проводилась. Локальный headless Chrome на этой машине
показывал искажённую вёрстку при явном window-size ~390px (обрезание текста), но это
воспроизвелось даже на тривиальной тестовой HTML-странице без единой строки моего кода —
подтверждённый артефакт headless-окружения (вероятно, ограничение окна на уровне macOS), а не
баг сайта. При ширине 600px и выше вёрстка стабильна и корректна на всех проверенных точках
останова (720px, 800px, 860px). **Рекомендация: проверить на реальном телефоне перед запуском
рекламы** — это единственное, что нельзя было верифицировать в данном окружении.

## Аудит доступности (a11y)
- [x] Контраст текста — **найдена и исправлена реальная проблема**: акцентный цвет `--putty`
  (#B0876A) для мелких эйброу-подписей на светлом фоне давал ~2.46:1 (нужно 4.5:1 для обычного
  текста по WCAG AA). Добавлен `--putty-ink` (#7A5240, ~5.2:1 на фоне `--sand`/`--plaster`) для
  всех эйброу на светлых секциях; исходный `--putty` оставлен только на тёмных секциях
  (`.band-ink .eyebrow`), где он и так проходит AA (~4.77:1). Также `--muted-2` (использовался для
  подписей под цифрами и меток формы) был затемнён с #9A8E7E (~2.45:1, не проходил) до #675C50
  (~4.98:1, проходит).
- [x] Все интерактивные элементы доступны с клавиатуры — ссылки/кнопки/`<details>`/поля формы
  нативные, добавлен `:focus-visible` стиль и skip-link «Перейти к содержимому»
- [x] Видимые фокус-индикаторы — добавлены через `:focus-visible{outline:2px solid var(--putty)}`
- [x] `prefers-reduced-motion` — учтено и в CSS (реveal без transform/opacity-анимации), и в JS
  (GSAP/ScrollTrigger и parallax не инициализируются при reduced motion)
- [x] Семантическая разметка — `header/nav/main/section aria-labelledby/footer`, `<ol>` для шагов,
  `scope="col"` в таблице сравнения
